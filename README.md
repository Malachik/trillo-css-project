# 🏨 Trillo CSS Project

A modern, responsive hotel booking UI built using **HTML5, SCSS (Sass), and Flexbox**.
This project focuses on mastering real-world layout techniques and clean UI structure.

---





---


---

## 🛠️ Built With

* HTML5
* SCSS (Sass)
* CSS Flexbox
* CSS Variables
* SVG Icons (sprite system)
* Responsive Design

---

## 📁 Project Structure

trillo-css-project/
│
├── css/              # Compiled CSS
├── sass/             # SCSS source files
│   ├── _base.scss
│   ├── _components.scss
│   ├── _layout.scss
│   └── main.scss
│
├── img/              # Images & SVG sprite
├── index.html        # Main HTML file
├── package.json      # Node scripts & dependencies
└── README.md

---

## ⚙️ Setup & Usage

### 1. Install dependencies

npm install

### 2. Run SCSS in watch mode

npm run watch:sass

### 3. Build CSS for production

npm run build:sass

---

## 🎯 Features

* Fully responsive layout
* Modern Flexbox-based design
* SCSS architecture (modular structure)
* SVG icon system using sprite file
* Interactive UI elements (hover, focus states)
* Clean and reusable components

---

## 📌 Notes

* SCSS compiles into `css/style.css`
* Do NOT edit compiled CSS directly
* Use `.gitignore` to exclude `node_modules`
